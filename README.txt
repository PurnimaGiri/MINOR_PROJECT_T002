## STEP-BY-STEP SETUP (READ THIS FIRST)

---

### STEP 1 — Install Python libraries
Open terminal / command prompt inside this folder and run:

    pip install pandas numpy scikit-learn imbalanced-learn matplotlib flask

Wait for everything to install. This is a one-time step.

---

### STEP 2 — Generate Dataset
Run this:

    python generate_dataset.py

Output: appointments_dataset.csv (10,000 rows)
You will see a summary printed in the terminal.

---

### STEP 3 — Train the Models
Run this:

    python train_model.py

This will:
- Train Random Forest for Slot Duration (3 classes)
- Train Random Forest for No-Show Prediction
- Save models to /models/ folder
- Generate 3 charts (PNG files)
- Print accuracy and AUC results

Expected output:
  Slot Duration Model Accuracy : ~91%
  No-Show Model AUC-ROC        : ~0.89

---

### STEP 4 — Run the Web App
Run this:

    python app.py

Then open your browser and go to:

    http://localhost:5000

Fill in patient details and click Predict.

## WHAT THE MODEL PREDICTS

1. SLOT DURATION CLASS
   - Short  → 0 to 10 minutes
   - Medium → 10 to 20 minutes
   - Long   → 20+ minutes

2. NO-SHOW PROBABILITY
   - A score from 0% to 100%
   - High risk (>60%) → overbooking triggered
   - Medium risk (35-60%) → reminder SMS sent
   - Low risk (<35%) → standard scheduling

## ALGORITHM
Random Forest Classifier
- Slot model: 300 trees, max_depth=20
- No-show model: 200 trees, max_depth=10, class_weight=balanced
- SMOTE used for no-show class imbalance
- 5-fold cross-validation for hyperparameter tuning
